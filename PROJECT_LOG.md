## 2026-03-14 — 프로젝트 킥오프

### 아이디어 요약
PC를 포맷한 후 사람들이 일일이 검색해서 설치해야 하는 기본 프로그램들의
**공식 다운로드 링크를 한 곳에 모아서 제공**하는 사이트.

### 경쟁 사이트 분석
| 사이트 | 방식 | 차별점 |
|--------|------|--------|
| Ninite.com | 자동설치 번들러 | 영어 전용, 한국 프로그램 없음 |
| PCTools (우리) | 공식 링크 허브 | 한국어, 한국 필수 프로그램 포함 |

### SEO 전략
**타겟 키워드:**
- 포맷 후 필수 프로그램 (월 검색량 높음)
- 윈도우 설치 후 해야할 것
- PC 기본 프로그램 추천
- 무료 프로그램 다운로드 모음
- [프로그램명] 다운로드 (개별 페이지)

**SEO 원칙:**
- 각 프로그램별 개별 페이지 (롱테일 키워드 공략)
- 카테고리별 페이지
- 메인 페이지: 포맷 후 체크리스트 형식
- 빠른 로딩 속도 (Core Web Vitals)
- 모바일 최적화

8. **드라이버** — 각 제조사별 링크
9. **메신저** — KakaoTalk, Discord, Zoom, Slack
10. **한국 필수** — 카카오톡, 네이버 웨일, 알약, 반디집

## 2026-03-14 — 기술/디자인 결정 확정

### 확정 사항
| 항목 | 결정 |
|------|------|
| 기술 스택 | 순수 HTML + CSS (PDFKit과 동일) |
| 언어 | 한국어 메인 |
| 링크 방식 | 공식 사이트 외부 링크 (파일 직접 호스팅 없음) |
| UI 레퍼런스 | pdfkit.wooahouse.com |

### UI 구조 (PDFKit 기반)
- 헤더: 로고 + 카테고리 네비게이션
- 히어로: 사이트 설명 + 핵심 메시지
- 본문: 카테고리별 카드 그리드 (emoji + 프로그램명 + 설명)
- 피처 배너: 안전/무료/최신 등 신뢰 지표
- 푸터: 카테고리별 전체 링크 목록

### 개발할 파일 목록
- `index.html` — 메인 허브 페이지
- `css/style.css` — PDFKit style.css 기반으로 커스터마이즈
- 각 카테고리별 상세 페이지 (브라우저, 보안, 압축, 미디어 등)
- `about.html`, `privacy.html`

### AdSense 배치 전략
- 히어로 아래 상단 광고 1개
- 카테고리 사이 중간 광고 1~2개
- 푸터 위 하단 광고 1개
- 각 카테고리 상세 페이지: 콘텐츠 상단/하단
